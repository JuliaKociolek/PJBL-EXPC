# Sistema de Controle de Projetos Acadêmicos

Projeto desenvolvido por **Julia Machado Kociolek**

## 📌 Descrição

Sistema web para gerenciamento de projetos acadêmicos, permitindo operações de CRUD (Criar, Ler, Atualizar e Excluir).

## 🚀 Tecnologias utilizadas

- React
- Node.js
- Express
- MySQL
- Axios

## ⚙️ Como executar o projeto

### 🔹 Backend
```bash
cd backend
npm install
node server.js
🔹 Frontend
cd frontend
npm install
npm start
🗄️ Banco de dados
Importar o arquivo .sql no MySQL Workbench ou XAMPP
Certificar-se de que o banco está rodando
📌 Funcionalidades
Criar projetos
Listar projetos
Editar projetos
Visualizar detalhes
